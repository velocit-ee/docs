# Code of conduct

velocitee follows the
[velocit-ee organisation Code of Conduct](https://github.com/velocit-ee/.github/blob/main/CODE_OF_CONDUCT.md).

In short: be kind, assume good faith, and remember that everyone reading
your message had a different morning than you. Disagreements are normal;
disrespect is not.

For violations: contact `inquiries@velocit.ee`. Reports are handled
privately.
